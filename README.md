# YOLOV8-CS2 v2

基于 YOLOv8 ONNX 推理的 CS2 视觉自动化实验项目，v2 版本完成了底层架构重构，引入多进程分离、状态机追踪与压枪宏系统。

> 本项目面向算法学习、工程验证和本地实验环境。请在遵守相关法律法规、平台规则和软件服务条款的前提下使用。

---

## v2 新特性

- **多进程架构**：视觉观测进程与控制输出进程完全分离，共享内存原子缓冲区通信
- **状态机追踪**：四状态闸门（搜索 → 确认 → 锁定 → 丢失），彻底解决目标跳变问题
- **匈牙利目标匹配 + 卡尔曼滤波**：跨帧稳定追踪同一目标，抑制检测框抖动
- **压枪宏系统**：内置 AK47 / 加利尔 / SG553 / M4A4 / M4A1 / AUG 弹道补偿数据
- **新模型**：`cs2警0胸1头匪2胸3头.onnx`，10万数据集，四分类（警胸/警头/匪胸/匪头）
- **死斗模式**：切换后对所有目标不区分阵营
- **参数热更新**：所有调参通过 GUI 滑块实时生效，无需重启

```

├── cs2警0胸1头匪2胸3头.onnx          # ONNX 模型文件
├── lj.dll                            # 外部动态链接库
├── aim_config.json                   # 参数配置文件
└── .gitignore
```

## 环境要求

- Windows 10 / Windows 11（64位）
- Python 3.9+（64位）
- 支持 DirectML、CUDA 或 CPU 的运行环境

安装依赖：

```bash
pip install opencv-python numpy onnxruntime-directml pywin32 scipy
```

不使用 DirectML 时：

```bash
pip install onnxruntime
```

## 运行方式

```bash
python 优化2.py
```

运行前确认：
- `cs2警0胸1头匪2胸3头.onnx` 与 `lj.dll` 位于同一目录
- Python 环境为 64 位
- 如 DLL 加载失败，请以管理员身份运行

## 配置文件

`aim_config.json` 保存当前参数，支持通过 GUI 实时调整并保存：

| 字段 | 说明 |
|------|------|
| `smooth_x` | X 轴平滑增益 |
| `smooth_y` | Y 轴平滑增益 |
| `max_disp` | 每帧最大位移限制（像素） |
| `conf` | 检测置信度阈值 |
| `aim_part` | 瞄准部位比例（0.1偏头 / 0.3偏胸） |
| `deadzone` | 死区半径，低于此值停止输出 |
| `kp` | PID 比例系数 |
| `kd` | PID 微分系数 |

## 快捷键

| 按键 | 功能 |
|------|------|
| F8 | 挂起 / 唤醒自瞄 |
| F7 | 关闭压枪宏 |
| 中键 (G3) | 切换 AK47 / 加利尔 / SG553 |
| 上侧键 (G5) | 切换 M4A4 / AUG |
