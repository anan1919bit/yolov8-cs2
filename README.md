# YOLOV8-CS2

基于 YOLOv8 与 PID 控制的 Windows 视觉自动化测试系统。项目集成了屏幕目标检测、闭环反馈控制、仿生运动轨迹和参数微调界面，主要用于计算机视觉、实时检测、控制算法和本地自动化测试研究。

## 项目简介

程序通过 YOLOv8 ONNX 模型捕获屏幕区域中的目标，并结合动态 PID 算法、坐标防抖、目标粘滞和贝塞尔曲线轨迹，输出更平滑的鼠标移动控制。项目同时提供打包后的参数配置界面，便于在不同测试场景下快速调整控制参数。

## 功能界面

打包后的界面名称为 `System Host Process 配置`，核心区域是“物理引擎参数微调”面板。界面支持：

- 预设模式切换，例如 `单体死锁稳健模式`
- 远距离爆发拉力调节
- 近距离磁吸倍率调节
- 准星急停防抖调节
- 模型识别防抖调节
- 绝对静止死区调节
- 拉枪拟真度调节
- 视觉坐标平滑调节
- 目标强锁半径调节
- 目标切换延迟调节
- 锁头偏移比例调节
- `F8` 自瞄主开关
- 参数修改后自动安全保存

## 核心技术栈

- 目标检测：YOLOv8 + ONNX Runtime
- 推理加速：DirectML / CUDA / CPU 自动选择
- 屏幕捕获：Windows API
- 控制算法：动态 PID 控制
- 运动轨迹：贝塞尔曲线插值
- 图形界面：CustomTkinter
- 底层调用：Python `ctypes` 调用外部键鼠驱动库

## 核心特性

- 动态微观磁吸：针对近距离目标动态调整控制强度，减少末端误差。
- 空间锚点防抖：过滤检测框抖动导致的高频坐标变化。
- 绝对静止死区：进入阈值范围后清空惯性与历史误差，降低细微抖动。
- 仿生平滑轨迹：通过贝塞尔曲线生成更自然的移动路径。
- 多参数配置：通过 `settings.json` 保存当前方案与参数。
- 图形化调参：运行或打包后可直接通过界面调整主要参数。

## 环境要求

- Windows 10 / 11
- Python 3.9 或更高版本
- 64 位 Python 环境
- 可选：支持 DirectML 或 CUDA 的 GPU

建议安装依赖：

```powershell
pip install opencv-python numpy onnxruntime-directml pywin32 customtkinter
```

如果不使用 DirectML，可根据环境改用：

```powershell
pip install onnxruntime
```

## 运行方法

进入项目目录：

```powershell
cd "D:\yolov8traintool\第二次打包"
```

```powershell

```

由于项目涉及 Windows 底层接口和外部动态链接库，运行源码或打包后的程序时，建议使用管理员身份启动。

## 配置说明

`settings.json` 用于保存当前参数方案和控制参数，例如：

- `kp`：PID 比例系数
- `kd`：PID 微分系数
- `平滑`：移动平滑参数
- `死区`：小偏移忽略范围
- `视觉平滑`：检测目标坐标平滑参数
- `粘滞半径`：目标保持范围
- `切换延迟时间`：目标切换延迟
- `锁头比例`：目标高度偏移比例
- `微观磁吸`：近距离修正强度
- `坐标防抖`：坐标抖动过滤阈值

不同设备、模型和测试场景下参数表现会有差异，建议先使用保守参数进行本地验证，再逐步微调。
