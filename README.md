# Vision-Tracker: 基于 YOLOv8 与 PID 控制的视觉自动化测试系统

## 项目简介
本项目是一个集成了计算机视觉目标检测、闭环反馈控制与仿生运动轨迹的自动化追踪系统。程序通过 YOLOv8 实时捕获屏幕目标，并结合动态 PID 算法与贝塞尔曲线，计算并输出平滑、精准的物理硬件级鼠标操控指令。

## 核心技术栈
* **目标检测:** YOLOv8 (基于 ONNX Runtime 进行 DirectML/CUDA 加速)
* **控制理论:** 动态增益 PID 控制器、卡尔曼滤波 (Kalman Filter)
* **运动学:** 贝塞尔曲线 (Bézier curve) 轨迹插值
* **图形界面:** CustomTkinter (支持异步无感保存与多配置管理)
* **底层通信:** Python ctypes 封装调用硬件级键鼠驱动

## 核心物理引擎特性
* **动态微观磁吸 (Dynamic Micro-Magnet):** 建立连续缓动曲线，针对近距离目标自动非线性放大 Kp 增益，解决传统 PID 的末端静差问题。
* **空间锚点防抖 (Spatial Hysteresis):** 引入空间滞后与坐标冻结逻辑，强行过滤 YOLO 视觉模型自带的像素级高频边框闪烁。
* **绝对静止死区 (Absolute Deadzone):** 准星进入物理阈值后瞬间清空滑动惯性与 PID 历史积分缓存，实现防抖死锁。
* **仿生平滑轨迹:** 融合动态平滑因子与贝塞尔曲线，消除自动化指针移动的机械感与突变加速度。

## 运行与部署
请确保项目根目录或打包后的可执行文件同级目录下，包含以下关键资产文件：
1. YOLOv8 模型权重文件 (例如 `cs2-10万数据集四个目标.onnx`)
2. 硬件驱动动态链接库 (`lj.dll`)

**启动说明：**
由于涉及底层硬件驱动的加载与调用，运行源码或编译后的可执行程序时，务必使用**管理员身份**启动。
